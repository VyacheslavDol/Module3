Задание 3.1 Модуль 3
При запуске general необходиом передать аргумент запуска - количество чисел, которое будет отправлено.
В general программе создается очередь, в которую с некоторым интервалом поступают случайные числа
В subor программе происходит подключение к очреди, созданной general программой, происходит считывание сообщений и вывод их на экран, при получении числа 255, программа удаляет очередь.
Для проверки работы можно воспользоваться двумя терминалами
